# Калькулятор квалификации S1 в Greenway

Интерактивный сайт-калькулятор для расчёта стратегии закрытия квалификации S1/S2.

## Что умеет сайт

- ввод личного объёма;
- добавление до 20 кружочков первой линии;
- автоматический расчёт группового объёма;
- расчёт подарочного и денежного счёта;
- расчёт бонуса наставника и премии наставника;
- отображение S1 при 750 PV и S2 при 1500 PV;
- конфетти при закрытии S1;
- генерация картинки 9:16 с текущей стратегией.

## GitHub Pages

Файл `index.html` лежит в корне репозитория и готов для публикации через GitHub Pages.
